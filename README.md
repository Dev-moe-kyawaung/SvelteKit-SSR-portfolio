# Portfolio SvelteKit SSR

A premium SvelteKit SSR portfolio starter with luxury visuals, SEO, RSS, sitemap, contact form, and CI-ready structure.

## Features
- SvelteKit SSR
- Premium dark-gradient UI
- SEO metadata
- RSS, sitemap, robots
- Contact form action
- Tailwind styling
- Docker + GitHub Actions
- Test folders for unit, integration, and e2e

## Development
```bash
pnpm install
pnpm dev
